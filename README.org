** Git

*** Prepare

#+begin_src shell-script
sudo aptitude build-dep git
#+end_src

*** Download and install

https://github.com/git/git

#+begin_src shell-script
make configure
./configure --prefix=/usr/local
make all doc
sudo make install install-doc install-html
#+end_src

** Emacs

*** Prepare

#+begin_src shell-script
sudo aptitude build-dep emacs
#+end_src

*** Download and install

https://savannah.gnu.org/projects/emacs/

http://www.gnu.org/software/emacs/

#+begin_src shell-script
./configure --prefix=/usr/local
# To support 'modules': vterm, etc.
# NOTE: should install: libvterm-dev (Debian, Ubuntu)
./configure --with-modules --prefix=/usr/local
make -j$(nproc)
sudo make install
#+end_src

*** Ctags

**** Download and install

https://github.com/universal-ctags/ctags ([[https://github.com/universal-ctags][universal-ctags]])

#+begin_src shell-script
git clone https://github.com/universal-ctags/ctags.git
cd ctags
# Get last Git tag from the master branch
git checkout $(git describe --tags --abbrev=0 master)
./autogen.sh
# Set path to the ctags (/usr/local/bin/ctags)
./configure --prefix=/usr/local
make -j$(nproc)
sudo make install
#+end_src

**** Add alternatives

If you will need alternative current the Emacs ctags, your may add it by the way:

#+begin_src shell-script
# Copy
sudo cp /usr/local/bin/ctags /usr/local/bin/ctags-universal-<version>
# Add alternatives
sudo update-alternatives --install /usr/local/bin/ctags ctags /usr/local/bin/ctags-universal-<version> 9
#+end_src

**** Installation for Windows OS

[[https://github.com/universal-ctags/ctags][Universal Ctags]] - copy binary to PATH directory (download and unpack to c:\emacs\bin, overwrite default ctags.exe)

*** Stow - software package installation manager

[[https://linuxconfig.org/how-to-use-gnu-stow-to-manage-programs-installed-from-source-and-dotfiles][GNU Stow to manage programs installed from source and dotfiles]]

#+begin_src shell
# 1. install stow
sudo aptitude install stow
# 2. build and install <package_name> to 'stow' directory (use prefix options):
./configure --prefix=/usr/local/stow/<package_name>
sudo make install
# 3. generate symlinks
cd /usr/local/stow
sudo stow --verbose=2 <package_name>
# 4.1. check links
stow -vv <package_name>
# 4.2. check that links replace executable files
tree /usr/local/bin/<package_name>
# 5. remove <package_name>
cd /usr/local/stow
sudo stow --delete <package_name>
#+end_src
